## Projet 3 – Sophie Bluel

### Formation Développeur Web (Diplôme de niveau 5 (bac +2)) – OpenClassrooms

#### Contexte:
En tant que développeur front-end, je dois aider mon équipe à finaliser le nouveau site internet d’une architecte, Sophie Bluel, grâce à la création d’un site dynamique avec JavaScript, communiquant avec une API.

#### Missions:
Développer:
- la page de présentation des travaux de l'architecte (à partir du HTML fourni) ;
- la page de connexion de l'administrateur du site (le client) (code à créer de zéro) ;
- la modale permettant d'uploader de nouveaux médias (code à créer from scratch).

#### Compétences acquises à l'issu du projet:
- Récupérer les données utilisateurs dans le JavaScript via des formulaires
- Gérer les événements utilisateurs avec JavaScript
- Manipuler les éléments du DOM avec JavaScript
